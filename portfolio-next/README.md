# Portfolio Next.js

Ce dossier contient une version moderne du portfolio développée avec Next.js.

## Démarrer localement

```bash
cd portfolio-next
npm install
npm run dev
```

## Déployer sur Vercel

1. Créez un dépôt GitHub.
2. Poussez ce dossier.
3. Connectez le dépôt à Vercel.
4. Déployez automatiquement.
